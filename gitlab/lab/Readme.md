## Configurar usuario para usar no Git

```bash
git config --global user.name "Rafael Mello" # Define o nome de usuario associado a todos os commits
```

```bash
git config --global user.email "rmellonh@gmail.com" # Define o e-mail usado para assinar os commits
```

## Comandos 

```bash
git config --list # Lista todas as configuracoes do Git consideradas no contexto atual
```

```bash
git config --global --list # Lista apenas as configuracoes salvas no escopo global (~/.gitconfig)
```

```bash
git init # Inicializa um novo repositorio Git no diretorio atual
```
